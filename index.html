CSCE 242: Website Applications

Emilie LaManna

This course focuses on web technologies used for developing client-server applications. CSCE 242 will develop skills for learning how to design, implement, and manage web-based applications that support effective communication between clients and servers. The curriculum covers a variety of web development tools and techniques, including HTML, CSS, JavaScript, and server-side scritping.

Assignments

  1. Basic HTML
  2. Basic CSS
